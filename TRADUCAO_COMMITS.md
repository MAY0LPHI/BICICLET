### Opção 1: Usando git filter-branch (Não Recomendado para Histórico Publicado)

```bash
# AVISO: Isto reescreverá todo o histórico
# Crie um backup primeiro!
git clone --mirror https://github.com/MAY0LPHI/BICICLETARIO backup.git

# Use git filter-branch com um script de tradução
git filter-branch --msg-filter '
  read msg
  case "$msg" in
    "Initial plan") echo "Plano inicial" ;;
    "Add files via upload") echo "Adicionar arquivos via upload" ;;
    # ... adicione todas as outras traduções ...
    *) echo "$msg" ;;
  esac
' --tag-name-filter cat -- --branches --tags
```
